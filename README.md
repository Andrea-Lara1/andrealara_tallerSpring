Taller Spring Boot

El taller consiste en el desarrollo de una API REST usando Spring Boot, en la caul se implementan diferents ejericicos praticos
usando controllers, service, repository y model.

*Los datos se almacenan en ArrayList
*Al reiniciar la aplicación la información se pierde
-----------------------------
Punto 1
Usuario
La funcion de este proyecto es: Registrar, listar y eliminar usuarios
Este ejercicio no tiene usuarios creados como tal en el proyecto, asi que al ejecutar no tendra ningun usuario
por lo cual al ejecutar se deben registrar/crear el usuario. 

ejecutar en navegador: 
http://localhost:8080/swagger-ui/index.html#/usuario

Endpoints

Metodo Post
http://localhost:8080/swagger-ui/index.html#/usuario-controller/guardar

Metodo Get
http://localhost:8080/swagger-ui/index.html#/usuario-controller/listar

Metodo Delete
http://localhost:8080/swagger-ui/index.html#/usuario-controller/eliminar
